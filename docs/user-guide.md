# 班级优化积分系统使用说明

这是一份给老师使用的简明操作手册。按照下面的顺序操作，就可以完成班级建立、学生导入、积分管理和成长查看。

## 1. 系统可以做什么

- 创建和管理班级
- 添加学生，或用 Excel 批量导入学生
- 创建小组并分配组员
- 设置积分项目
- 给学生或小组加分、减分
- 查看统计分析、评分历史和成长花园

## 2. 第一次使用

### 2.1 注册账号

打开系统登录页后，点击`立即注册`，按页面提示填写：

- 用户名
- 姓名
- 密码
- 确认密码
- 学科

填写完成后，点击`立即注册`。

### 2.2 登录系统

返回登录页，输入：

- 用户名
- 密码

点击`登录`后，进入`我的班级`页面。

## 3. 推荐操作顺序

建议第一次使用时，按下面 5 步完成初始化：

1. 新建班级
2. 设置积分项目
3. 添加学生
4. 创建小组
5. 开始评分和查看成长

## 4. 具体操作

### 4.1 新建班级

进入`我的班级`页面后：

1. 点击右上角`新建班级`
2. 填写班级名称、年级、学年
3. 可选填写班级简介
4. 选择视觉主题：
   - `树苗主题`
   - `种菜主题`
5. 点击`创建班级`

创建成功后，会自动进入该班级页面。

### 4.2 设置积分项目

进入班级后，先点击上方的`积分项目`标签页。

操作方法：

1. 点击`添加项目`
2. 填写项目名称
3. 选择类型：
   - `加分`
   - `减分`
4. 输入分值
5. 点击`确认添加`

建议先把常用项目都建好，例如：

- 积极发言
- 作业认真
- 纪律良好
- 未带作业
- 课堂提醒

说明：
如果还没有积分项目，后面给学生评分时会无法正常选择评分项。

### 4.3 添加学生

有两种方式：手动添加，或 Excel 批量导入。

#### 方式一：手动添加

在班级页的`学生管理`标签页中：

1. 点击`添加学生`
2. 填写姓名、学号
3. 可选填写性别、座位号
4. 点击`确认添加`

#### 方式二：Excel 批量导入

在班级页的`学生管理`标签页中：

1. 点击`导入`
2. 点击`下载 Excel 模板`
3. 按模板填写学生信息
4. 上传文件
5. 点击`预览`
6. 确认无误后点击`确认导入`

模板字段如下：

- `姓名`
- `学号`
- `性别`
- `座号`
- `小组名称`
- `备注`

填写说明：

- `姓名`和`学号`不能为空
- 同一个班级中，学号不能重复
- 如果填写了`小组名称`，系统会自动创建对应小组并把学生加入该小组
- 导入前请先看`预览`结果，红色错误信息需要先处理

### 4.4 创建小组和管理成员

进入班级页后，点击`小组管理`标签页。

#### 新建小组

1. 点击`添加小组`
2. 输入小组名称
3. 点击`确认添加`

#### 给小组添加成员

1. 在对应小组右侧点击`成员`
2. 在`添加成员`下拉框中选择学生
3. 点击`添加`

说明：
一个学生同一时间只能在一个小组里。如果下拉框里没有该学生，通常表示他已经在别的小组中。

### 4.5 给学生评分

进入班级页后，打开`学生管理`标签页。

操作方法：

1. 在学生列表中，直接点击该学生所在行
2. 在弹出的评分窗口中选择积分项目
3. 按需要调整分值
4. 可填写备注
5. 点击`确认`

评分完成后，学生积分会立即更新。

### 4.6 给小组评分

进入`小组管理`标签页后：

1. 找到对应小组
2. 点击`评分`
3. 输入分值
4. 可填写项目名称和备注
5. 点击`确认`

### 4.7 查看统计分析

进入班级页后，点击`统计分析`标签页，可以查看：

- 本周加分和减分
- 本月加分和减分
- 学生排名
- 小组排名

适合用来查看班级整体表现和阶段性变化。

### 4.8 查看评分历史

进入班级页后，点击`历史记录`标签页，可以查看每一次评分记录，包括：

- 时间
- 类型
- 评分对象
- 项目
- 分值
- 备注
- 状态

### 4.9 查看成长花园

在班级页顶部，点击`进入花园`。

进入后可以切换查看：

- `班级主景`
- `小组花园`
- `我的园地墙`

这里会用成长可视化方式展示班级、小组和学生的积分变化，更适合课堂展示和激励使用。

## 5. 个人中心

在`我的班级`页面右上角，点击`个人中心`，可以修改：

- 姓名
- 学科
- 手机
- 邮箱
- 个人简介

修改后点击`保存修改`即可。

如需退出账号，可点击`退出登录`。

## 6. 常见问题

### 6.1 为什么不能给学生评分？

先检查班级里是否已经添加了`积分项目`。如果没有积分项目，请先到`积分项目`标签页进行设置。

### 6.2 为什么导入失败？

常见原因有：

- 姓名为空
- 学号为空
- 学号重复
- 表格内容填写不规范

建议先点击`预览`，根据错误提示修改 Excel 后再重新导入。

### 6.3 为什么学生不能加入小组？

因为该学生可能已经在其他小组中。一个学生一次只能属于一个小组。

### 6.4 班级信息在哪里修改？

回到`我的班级`页面，在对应班级卡片上点击`编辑`，修改后保存即可。

## 7. 一句话版使用流程

注册/登录 -> 新建班级 -> 添加积分项目 -> 添加学生或批量导入 -> 创建小组 -> 开始评分 -> 查看统计和成长花园

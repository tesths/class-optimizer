# Farm Scene Asset Sources

This directory contains a curated subset of CC0 isometric farm assets prepared for the garden scene.

## Source 1: Kenney - Isometric Miniature Farm
- Site: https://kenney.nl/assets/isometric-miniature-farm
- Download used: `kenney_isometric-miniature-farm.zip`
- License: CC0 1.0 (see `KENNEY-LICENSE.txt`)
- Usage in this repo:
  - Base terrain tiles
  - Farmland and plank path tiles
  - Crop props (corn, hay)
  - Farm props (shack parts, fences, sacks, ladder, chimney)

## Source 2: OpenGameArt - Pixel farm and shack
- Page: https://opengameart.org/content/pixel-farm-and-shack
- File used: `crops_tileset.png`
- Local copy: `../crops/oga_crops_tileset.png`
- License listed on source page: CC0 1.0
- Attribution note on source page: "Credit pixel32 if you feel like it."

# 四、里程碑验收节点

## 里程碑 M1：可登录 + 可建班级
验收标准：
- 注册登录可用
- 教师资料可编辑
- 可新建、查看、编辑班级

## 里程碑 M2：可管理学生 + 可打分
验收标准：
- 可新增/编辑/删除学生
- 可配置积分项目
- 可给学生打分
- 可查看学生积分变化

## 里程碑 M3：可管理小组 + 可统计
验收标准：
- 可新建小组并分配成员
- 可给小组打分
- 可查看排行榜和统计图表

## 里程碑 M4：可追溯 + 可导入 + 可移动端使用
验收标准：
- 历史记录完整
- 管理员可撤销
- Excel 导入可用
- 手机端主要流程可用

# 五、当前建议你最后确认的事项
1. 登录方式固定用户名登录
2. 成长风格按班级统一
3. 导入时若小组不存在则自动创建
4. 协同教师可删除学生但不可删班级
